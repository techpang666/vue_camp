## **项目食用方式🍖**
```bash
# 拉取项目
git clone https://github.com/techpang666/vue_camp.git
cd vue_camp

# 切换淘宝源及下包
npm config set registry=https://registry.npm.taobao.org/
npm i

# 起飞(编译)项目🚀
npm run serve
```
